# 新闻播报配置（JST）

来源：工作区已落地配置整理版。

## 固定执行时间
- 09:00
- 17:00

## 固定取材时间窗口
### 09:00 播报
- 前一日 17:00 ～ 当日 09:00

### 17:00 播报
- 当日 09:00 ～ 当日 17:00

## 固定分区
- 日本
- 中国
- 国际

## 强约束
1. 正式播报必须对应 09:00 或 17:00 固定锚点。
2. 必须使用对应固定窗口，不得改成浮动“过去若干小时”。
3. 临时测试窗口不得污染正式播报窗口。
4. 临时额外播报必须明确标记为“测试”或“临时窗口”。

## 排版强约束
1. 只允许一级数字标题：1. 日本 / 2. 中国 / 3. 国际
2. 一级标题下必须且只空 1 行
3. 子层级禁止数字编号
4. 主区块之间必须且只空 1 行

## 当前状态
- 正常新闻播报暂停
- 在格式化解决方案真正落地并通过检查前，只允许进行实际数据测试（非冒烟）
