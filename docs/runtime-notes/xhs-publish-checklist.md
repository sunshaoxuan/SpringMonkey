# 小红书发布前后检查清单

发布前必须完成：

1. 在 Google Docs 中确认文案和图片为最终版。
2. 上传到小红书编辑器后，检查图片数量、顺序、封面预览。
3. 检查标题与正文：无商品番号、无解释选题语气、无 AI 卡片格式。
4. 检查话题：
   - 没有粘连错字。
   - 没有被截断成短词。
   - 没有残留测试字符或多余井号。
   - 平台预览里的显示能接受。
5. 如果要求每条话题一行且无空行，必须通过小红书自己的话题控件逐个选择完整话题，并在预览验证；不能直接粘贴赌格式。
6. 只有预览确认无问题后，才点击发布。

发布后必须完成：

1. 到笔记管理确认新笔记存在。
2. 确认状态为审核中/已发布，而不是草稿。
3. 复核标题和可见正文片段。
4. 如发现格式错误，先判断能否安全编辑；不要在已发布笔记上做未知实验。
